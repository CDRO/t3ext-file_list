Add file_list by TypoScript
"""""""""""""""""""""""""""

If EXT:file_list should be integrated by using TypoScript only, you can use this
code snippet:

.. code-block:: typoscript

    lib.filelist = USER
    lib.filelist {
        userFunc = TYPO3\CMS\Extbase\Core\Bootstrap->run
        extensionName = FileList
        vendorName = Causal
        pluginName = Filelist

        switchableControllerActions {
            File {
                1 = list
            }
        }

        settings < plugin.tx_filelist.settings
        settings {
            path = file:1:/path/to/directory/
            mode = FOLDER
            includeSubfolders = 1
            templateLayout = ThumbnailDescription
        }
    }

Now you can use the object lib.filelist.
